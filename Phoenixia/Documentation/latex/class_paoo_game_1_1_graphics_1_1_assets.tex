Clasa incarca fiecare element grafic necesar jocului. 

\mbox{\hyperlink{class_paoo_game_1_1_game}{Game}} assets include tot ce este folosit intr-\/un joc\+: imagini, sunete, harti etc.

Functia initializaza referintele catre elementele grafice utilizate. 

Aceasta functie poate fi rescrisa astfel incat elementele grafice incarcate/utilizate sa fie parametrizate. Din acest motiv referintele nu sunt finale. Se creaza temporar un obiect \mbox{\hyperlink{class_paoo_game_1_1_graphics_1_1_sprite_sheet}{Sprite\+Sheet}} initializat prin intermediul clasei \mbox{\hyperlink{class_paoo_game_1_1_graphics_1_1_image_loader}{Image\+Loader}}

Se obtin subimaginile corespunzatoare elementelor necesare în joc.

Se obtin subimaginile corespunzatoare elementelor necesare nivelului 2.

Se obtin subimaginile corespunzatoare butoanelor necesare.

Se obtin subimaginile corespunzatoare animațiilor regăsite în state-\/urile ABOUT, WINSTATE2.
