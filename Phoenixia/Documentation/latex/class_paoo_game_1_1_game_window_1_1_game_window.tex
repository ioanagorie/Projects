\doxysubsection{Detailed Description}
Implementeaza notiunea de fereastra a jocului. 

Membrul wnd\+Frame este un obiect de tip JFrame care va avea utilitatea unei ferestre grafice si totodata si cea a unui container (toate elementele grafice vor fi continute de fereastra).

Avand in vedere ca dimensiunea ferestrei poate fi modificata si corespunzator continutul actualizat (aici ma refer la dalele randate) va recomand sa constrangeti deocamdata jucatorul sa se joace in fereastra stabilitata de voi. Puteti reveni asupra urmatorului apel ulterior.

Recomand ca fereastra sa apara in centrul ecranului. Pentru orice alte pozitie se va apela \char`\"{}wnd\+Frame.\+set\+Location(x, y)\char`\"{} etc.
