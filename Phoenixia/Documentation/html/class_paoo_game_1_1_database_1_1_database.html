Detailed Description

Clasa ce implementeaza notiunea de baza de date a jocului.

Prin intermediul bazei de date care se va conecta la program, userul va avea posibilitatea sa isi reia progresul in joc de la momentul in care l-a salvat. Design patternul folosit pentru implementarea conexiunii programului cu baza de date este Singleton.
