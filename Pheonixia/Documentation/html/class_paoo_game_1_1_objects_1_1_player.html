Detailed Description

Implementeaza notiunea de player/jucator (personajul controlat de jucator).

Elementele suplimentare pe care le aduce fata de clasa de baza sunt: imaginea (acest atribut poate fi ridicat si in clasa de baza) grafica vietile dreptunghiul de coliziune
