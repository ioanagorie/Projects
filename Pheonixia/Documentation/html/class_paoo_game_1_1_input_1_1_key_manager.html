Detailed Description

Gestioneaza intrarea (input-ul) de tastatura.

Clasa citeste daca au fost apasata o tasta, stabiliteste ce tasta a fost actionata si seteaza corespunzator un flag. In program trebuie sa se tina cont de flagul aferent tastei de interes. Daca flagul respectiv este true inseamna ca tasta respectiva a fost apasata si false nu a fost apasata.
