Detailed Description

Clasa responsabila de gestionarea UI.

Aici se retin toate obiectele de tip UIObject din joc. Clasa contine metode existente in UIObject (Update, Draw, MoveMouse, MouseRelease) in care se apeleaza pentru toate obiectele retinute aceeasi metoda, plus inca doua metode de intretinere (addObj si deleteObj);
