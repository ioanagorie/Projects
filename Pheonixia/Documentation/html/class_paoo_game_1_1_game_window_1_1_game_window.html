Detailed Description

Implementeaza notiunea de fereastra a jocului.

Membrul wndFrame este un obiect de tip JFrame care va avea utilitatea unei ferestre grafice si totodata si cea a unui container (toate elementele grafice vor fi continute de fereastra).

void PaooGame.GameWindow.GameWindow.BuildGameWindow ( )

Daca fereastra a mai fost construita intr-un apel anterior se renunta la apel

Aloca memorie pentru obiectul de tip fereastra si seteaza denumirea ce apare in bara de titlu.

Seteaza dimensiunile ferestrei in pixeli

Operatia de inchidere (fereastra sa poata fi inchisa atunci cand este apasat butonul x din dreapta sus al ferestrei). Totodata acest lucru garanteaza ca nu doar fereastra va fi inchisa ci intregul program

Avand in vedere ca dimensiunea ferestrei poate fi modificata si corespunzator continutul actualizat (aici ma refer la dalele randate) va recomand sa constrangeti deocamdata jucatorul sa se joace in fereastra stabilitata de voi. Puteti reveni asupra urmatorului apel ulterior.

Recomand ca fereastra sa apara in centrul ecranului. Pentru orice alte pozitie se va apela "wndFrame.setLocation(x, y)" etc.

Implicit o fereastra cand este creata nu este vizibila motiv pentru care trebuie setata aceasta proprietate

Creaza obiectul de tip canvas (panza) pe care se poate desena.

In aceeasi maniera trebuiesc setate proprietatile pentru acest obiect canvas (panza): dimensiuni preferabile, minime, maxime etc. Urmotorul apel de functie seteaza dimensiunea "preferata"/implicita a obiectului de tip canvas. Functia primeste ca parametru un obiect de tip Dimension ca incapsuleaza doua proprietati: latime si inaltime. Cum acest obiect nu exista a fost creat unul si dat ca parametru.

Avand in vedere ca elementele unei ferestre pot fi scalate atunci cand fereastra este redimensionata

Avand in vedere ca obiectul de tip canvas, proaspat creat, nu este automat adaugat in fereastra trebuie apelata metoda add a obiectul wndFrame

Urmatorul apel de functie are ca scop eventuala redimensionare a ferestrei ca tot ce contine sa poate fi afisat complet
