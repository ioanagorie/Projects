\doxysubsection{Detailed Description}
Defineste notiunea abstracta de personaj/caracter din joc. Obstacolele din joc sunt considerate a fi Personaje, întrucât ele reprezintă o abstractizare a notiunii de inamic, deoarece coliziunea lor cu player-\/ul are un impact asupra acestuia. 

Notiunea este definita de viata, viteza de deplasare si de inaltimea si larimea imaginii care il defineste.
