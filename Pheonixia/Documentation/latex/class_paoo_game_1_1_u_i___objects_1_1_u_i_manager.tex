\doxysubsection{Detailed Description}
Clasa responsabila de gestionarea UI. 

Aici se retin toate obiectele de tip \mbox{\hyperlink{class_paoo_game_1_1_u_i___objects_1_1_u_i_object}{UIObject}} din joc. Clasa contine metode existente in \mbox{\hyperlink{class_paoo_game_1_1_u_i___objects_1_1_u_i_object}{UIObject}} (Update, Draw, Move\+Mouse, Mouse\+Release) in care se apeleaza pentru toate obiectele retinute aceeasi metoda, plus inca doua metode de intretinere (add\+Obj si delete\+Obj);
