\doxysubsection{Detailed Description}
Gestioneaza intrarea (input-\/ul) de la mouse. 

Clasa citeste daca au fost apasat un click si seteaza corespunzator un flag. In program trebuie sa se tina cont de flagul aferent click-\/ului de interes. Daca flagul respectiv este true inseamna ca click-\/ul respectiv a fost apasat, iar daca este false, nu a fost apasat.
